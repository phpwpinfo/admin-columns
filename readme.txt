Add custom admin columns to post types to provide sorting and filtering through these extra columns
== Description ==

**Features**

1. Ability to add custom columns per post type,
2. Ability to sort posts in the admin panel through the custom columns,
3. Ability to filter posts in the admin panel through the custom columns,

For detailed instructions on using this plugin visit: http://wordpressindex.com/wordpress-admin-custom-columns-sorting-and-filtering/




== Installation ==

1. Upload the plugin files Google Maps Plotting in the same way you'd install any other plugin.
2. Activate the plugin
